## Synopsis

Common Swift is a set of shared foundations like CommonShell and CommonProcess that make
subprocess execution predictable, typed, and portable across tooling.

## Status

- **Stage:** Active development
- **Surface:** Swift packages

## Focus

- Typed options and reliable process execution
- Portable adapters with strong defaults
- Clear API boundaries for CLI tooling
