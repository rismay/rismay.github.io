  Shared color system rules across all brands.
  ## Palette model
  - Each brand defines a primary palette and a neutral palette.
  - Primary colors communicate identity; neutrals carry most body content.
  - Accent colors are optional and limited to one per brand.
  ## Default neutral scale
  - **Ink 1000:** `#0A0A0A`
  - **Ink 900:** `#111111`
  - **Ink 800:** `#1A1A1A`
  - **Ink 700:** `#242424`
  - **Gray 600:** `#3A3A3A`
  - **Gray 500:** `#5A5A5A`
  - **Gray 400:** `#7A7A7A`
  - **Gray 300:** `#A0A0A0`
  - **Gray 200:** `#C8C8C8`
  - **Gray 100:** `#E5E5E5`
  - **Mist 50:** `#F4F4F4`
  - **Mist 0:** `#F9F9F9`
  ## Contrast
  - Body text must meet 4.5:1.
  - Large text may use 3:1 but prefer 4.5:1.
  - Avoid low-contrast color-on-color unless used as decoration.
  ## Usage rules
  - Use one primary color per section or surface.
  - Neutrals anchor background and typography.
  - Do not use gradients unless explicitly documented in a brand page.
}
