  Shared typography standards across all brands.
  ## Families
  - Each brand defines a display face and a text face.
  - Monospace is required for code and technical labels.
  ## Default type scale
  - **H1:** 64/72
  - **H2:** 40/48
  - **H3:** 28/36
  - **H4:** 22/30
  - **Body:** 18/28
  - **Small:** 14/22
  - **Code:** 14/22
  ## Usage
  - Typography establishes hierarchy before color or layout.
  - Avoid mixed typefaces within a single headline.
  - Keep line length between 55-75 characters.
  - Use consistent baseline rhythm.
}
