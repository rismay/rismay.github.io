  Shared asset rules across all brands.
  ## Formats
  - SVG for vector assets.
  - PNG for raster where required.
  - Keep file names kebab-case.
  ## Storage
- Store shared assets in the bundle `Resources/` directory.
- Reference assets by their bundle-relative name; avoid bundle name prefixes.
  - Keep source files alongside final exports when possible.
  ## Tokens
  - Provide a color swatch sheet for every palette.
  - Provide typography specimen images for each face.
  - Provide icon grid sheets for each icon set.
  ## Licensing
  - Record asset provenance and license in the brand page.
}
