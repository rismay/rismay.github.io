  Shared iconography rules across all brands.
  ## Style
  - Base grid: 24px or 32px
  - Stroke weight: 2px
  - Corner radius: 0-2px
  ## Usage
  - Prefer monochrome icons.
  - Use color sparingly, tied to the brand palette.
  - Avoid overly detailed icons at small sizes.
  ## Export
  - SVG with a 1x artboard.
  - Name files using kebab-case.
}
