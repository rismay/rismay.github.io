## Intent

Typography balances approachable readability with technical precision.

## Families

- Headings: SF Pro Display (fallback: SF Pro Text, system)
- Body: SF Pro Text (fallback: system)
- Code: SF Mono (fallback: Menlo, monospace)

## Scale and hierarchy

- H1: 32-36pt
- H2: 24-28pt
- H3: 20-22pt
- Body: 15-17pt
- Caption: 12-13pt

## Usage rules

- Reserve bold weights for headings and callouts.
- Avoid mixing too many weights on a single page.
- Code blocks use monospaced fonts with ligatures disabled.

## Do

- Keep line length between 60-90 characters for reading comfort.
- Use generous line height for documentation pages.

## Avoid

- All-caps headings for long sections.
- Overly small text in sidebars or inspector panels.
