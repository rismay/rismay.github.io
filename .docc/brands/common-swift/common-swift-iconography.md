## Intent

Icons should reinforce structure and navigation without competing with
content.

## Style

- Outline icons for navigation and categories.
- Filled icons only for active or selected states.
- Consistent stroke weight and corner radius across the set.

## Pictographs

- Library pictographs align to the core modules:
  - CommonShell
  - CommonCLI
  - CommonProcess

## Do

- Use a single icon style per surface.
- Align icons to a consistent baseline with text.

## Avoid

- Mixing outline and filled icons in the same navigation context.
- Using icons as the only cue for critical actions.
