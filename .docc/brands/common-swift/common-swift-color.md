## Intent

Define a clear, restrained palette that supports long-form reading and
technical credibility while keeping Swift Orange as the primary brand signal.

## Palette

- Primary: #F05138 (Swift Orange)
- Secondary: #333333 (Charcoal)
- Accent: #007AFF (Apple Blue)
- Background: #FFFFFF
- Surface: #F6F7F9
- Border: #E4E7EC
- Text primary: #1D1D1F
- Text secondary: #4B4B4B
- Success: #2E9D4B
- Warning: #B46A00
- Error: #D12C2C

## Usage rules

- Primary is reserved for calls-to-action and signature highlights.
- Accent is used for links, hover states, and focus rings.
- Secondary supports long-form text and section headers.
- Background and surface must maintain contrast >= 4.5.

## Do

- Use neutral surfaces for documentation pages.
- Keep color usage sparse and intentional.

## Avoid

- Overusing primary or accent for large backgrounds.
- Low-contrast combinations in long-form reading contexts.
