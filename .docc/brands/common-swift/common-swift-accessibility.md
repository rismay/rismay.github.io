## Intent

Accessibility is mandatory for reading and learning surfaces.

## Requirements

- Contrast ratio >= 4.5 for body text and UI controls.
- Keyboard-first navigation for doc navigation and code samples.
- Visible focus states that use the accent color.
- Reduce-motion preference disables non-essential animation.

## Do

- Provide readable font sizes and generous line height.
- Ensure interactive controls have descriptive labels.

## Avoid

- Relying on color alone to convey state.
- Small hit targets in navigation and inspector panels.
