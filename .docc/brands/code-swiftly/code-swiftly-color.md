## Intent

A baby-blue-led palette that signals clarity, calm, and trust while keeping
reading surfaces neutral and high contrast.

## Primary palette

- Primary blue: #4D9DE0 (Baby Blue)
- Deep blue: #0B3D91 (Navy anchor)
- Accent blue: #4285F4 (Google Blue)
- Guide blue: #0E67C6 (Chase-style blue)

## Supporting colors

- Background: #FFFFFF
- Surface: #F5F8FC
- Border: #E1E7F0
- Text primary: #121B2A
- Text secondary: #3B4A5E
- Success: #2E9D4B
- Warning: #B46A00
- Error: #D12C2C

## Usage rules

- Primary blue is used for calls-to-action and key highlights.
- Deep blue anchors headings and navigation.
- Accent blue is used for links, hover states, and focus rings.
- Background and surface must maintain contrast >= 4.5.

## Do

- Keep color usage sparse in long-form docs.
- Use blue gradients only in hero areas.

## Avoid

- Bright blue backgrounds behind long text.
- Mixing multiple blues in a single control.
