Define the assets standards for the Studio Laussat brand.

@Image(source: "brands-laussat-studio-laussat-studio-assets-hero", alt: "studio laussat: assets hero")

## Principles

- Grayscale assets only.
- Serif wordmarks use optical sizing, not manual distortion.
- Prefer quiet textures over gradients.

## Specifications

- **Icon:** 128x128, monochrome.
- **Card:** 640x400, grayscale palette.
- **Hero:** 1200x600, single focal element.
