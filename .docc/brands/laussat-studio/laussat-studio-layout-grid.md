Define the layout grid standards for the Studio Laussat brand.

@Image(source: "brands-laussat-studio-laussat-studio-layout-grid-hero", alt: "studio laussat: layout grid hero")

## Principles

- Wide margins and generous whitespace.
- Align to a single baseline grid for serif rhythm.
- Fewer columns; more breathing room.

## Specifications

- **Grid:** 6-column on desktop, 4-column on mobile.
- **Baseline:** 8px with 4px sub-steps.
