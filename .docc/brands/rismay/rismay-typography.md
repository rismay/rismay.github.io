  Geometric display + neutral text. Type is the primary visual system.
  ## Type stack
  - **Display:** Space Grotesk (600-700)
  - **Text:** IBM Plex Sans (400-500)
  - **Mono:** IBM Plex Mono (400)
  ## Hierarchy
  - **H1:** Space Grotesk 64/72, 700
  - **H2:** Space Grotesk 40/48, 600
  - **H3:** Space Grotesk 28/36, 600
  - **Body:** IBM Plex Sans 18/28, 400
  - **Small:** IBM Plex Sans 14/22, 400
  - **Code:** IBM Plex Mono 14/22, 400
  ## Rules
  - Prefer lowercase headings where possible.
  - Use tight letter spacing on large headings (-1% to -2%).
  - Keep line length between 55-75 characters.
}
