## Synopsis

A growing toolkit for CLI platforms, logging, configuration, and system utilities. These
libraries standardize how tools are built, shipped, and maintained across workstreams.

## Status

- **Stage:** Active development
- **Surface:** Swift packages + internal tooling

## Focus

- Consistent CLI ergonomics and observability
- Reliable config and environment handling
- Shared infrastructure for internal tools
