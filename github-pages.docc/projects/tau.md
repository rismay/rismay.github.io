## Synopsis

Tau is a macOS status bar app designed to keep daily flow visible and calm. The next evolution
adds options trading views and quick‑action workflows so market data stays lightweight and usable.

## Status

- **Stage:** Active development
- **Platform:** macOS (Catalyst)

## Focus

- Real‑time market glance views with minimal noise
- Fast capture and review for options positions
- Clear UI states that reduce context switching
